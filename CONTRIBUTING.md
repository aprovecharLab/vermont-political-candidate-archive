This is currently a solo project. If you find an error, omission, or have a source to suggest, please open an issue or submit a pull request. When contributing:

1. **Cite sources** — Every factual claim should be traceable to a publicly available source.
2. **Note dates** — Clearly mark when positions were stated or published.
3. **Flag uncertainty** — Use "not publicly stated" or "limited information available" rather than inferring.
4. **Maintain neutrality** — Present positions accurately; let readers draw conclusions.
5. **Follow templates** — Use the templates in `templates/` for consistency.
